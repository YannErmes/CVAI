### 4.4 PAGE SPEED OPTIMIZATION

**Core Web Vitals Targets (2025 Standards):**
- **LCP (Largest Contentful Paint)**: < 2.5 seconds (good)
- **FID (First Input Delay)**: < 100 milliseconds (good)
- **INP (Interaction to Next Paint)**: < 200 milliseconds (good)
- **CLS (Cumulative Layout Shift)**: < 0.1 (good)

**Optimization Checklist:**

1. **Image Optimization:**
   - Use WebP format with JPEG fallback
   - Compress all images (TinyPNG, ImageOptim)
   - Use proper image dimensions
   - Implement lazy loading for off-screen images
   - Use responsive images (srcset attribute)

2. **Code Splitting:**
   - Split React bundles by route
   - Load critical CSS inline
   - Defer non-critical JavaScript
   - Remove unused CSS with PurgeCSS (Tailwind does this)

3. **Caching Strategy:**
   - Set Cache-Control headers (1 year for static assets)
   - Use browser caching
   - Implement service workers for offline capability

4. **Content Delivery:**
   - Use CDN (Cloudflare, Akamai, etc.)
   - Enable GZIP compression
   - Minimize JavaScript bundle size

5. **Server Optimization:**
   - Use HTTP/2
   - Enable Brotli compression
   - Optimize server response time (TTFB < 600ms)

**Testing Tools:**
- Google PageSpeed Insights: https://pagespeed.web.dev/
- GTmetrix: https://gtmetrix.com/
- WebPageTest: https://www.webpagetest.org/
- Lighthouse (built into Chrome DevTools)

**Key Events to Track:**
1. `resume_created` - User completes resume
2. `template_selected` - User selects a template
3. `resume_downloaded` - User downloads resume
4. `form_started` - User begins resume
5. `form_abandoned` - User leaves mid-form
6. `ats_checker_used` - User uses ATS tool
7. `blog_post_read` - User reads blog (scroll depth)
8. `cta_clicked` - User clicks CTA button

**Implementation Example (Google Tag Manager):**
```javascript
// Track resume creation
gtag('event', 'resume_created', {
  'template': 'modern',
  'time_to_complete': 287, // seconds
  'device': 'mobile'
});

// Track downloads
```

- Improves article visibility
- Better social sharing preview
