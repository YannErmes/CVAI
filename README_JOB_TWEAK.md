## How It Works

### The Process
```
1. Read your current CV data
   â†“
2. Analyze the job posting you provided
   â†“
3. Identify relevant keywords & requirements
   â†“
4. Generate 2-3 specific improvements
   â†“
5. Show you before/after comparison
   â†“

   â†“

```

### What Gets Tweaked
- **Summary** - Reworded to match job requirements
- **Skills** - Reordered to show relevant skills first
- **Experience** - Reframed to highlight job-related achievements

### What Stays the Same
- Your name, email, phone, location
- Your education history
- Any fields not in the tweaks
- Your original CV (until you apply tweaks)

## FAQ

### Q: Does using Job Tweak cost my daily credits?
**A:** NO! Only full CV generation counts. Tweaks are unlimited.

### Q: What if I don't like the tweaks?
**A:** Click "Back" to regenerate with different tweaks, or "Discard" to close.

### Q: Can I apply only some tweaks?
**A:** YES! Uncheck any tweaks you don't want before clicking Apply.

### Q: How long does it take?
**A:** 2-3 seconds usually. Depends on job description length.

### Q: Will my original CV be changed?
**A:** Not until you click Apply. Click Discard to keep original.

### Q: Can I manually edit after tweaks?
**A:** YES! You can edit the form fields after applying tweaks.

### Q: Does it work with all templates?
**A:** YES! Tweaks work with every CV template.

### Q: What if I need API key?
**A:** Go to Settings, add your Gemini API key, try again.

### Q: Can I tweak the same CV multiple times?
**A:** YES! Create tailored versions for different jobs.

### Q: Is it really that easy to use?
**A:** YES! The feature is designed so anyone can use it instantly.
